Linear Regression Modeling

Founder CEO vs Hired CEO

Project Breif

Dataset is about performance of firms based on Founder CEO and Hired CEOs of 9 years with 2250 observations and 9 variables.

9 Variables

  1. Year: Each year the data observed by the CEO
  2. Founders' age: Founders' age
  3. Permid : CEO Id
  4. Volatility: Variability in performance
  5. TobinsQ: an actual performance metric
  6. Founder CEO:
  7. Lntobinq: Normalized observations of TobinsQ
  8. lnasset: Normalized observations of Asset
  9. Lnage: Normalized observations of age

Goal

To run linear regression model to find which kind of CEO performs better - Founder CEO vs Hired CEO

Result

Founder CEO showed better performance compared to Hired CEO

Univariate analysis: determining the mean of a subset of data where founder ceo = 1 or 0

Multivariate analysis: running linear regression modeling I found that it showed the same result of Founder CEOs
